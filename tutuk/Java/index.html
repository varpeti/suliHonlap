Java Developer Kit telepítése


sudo apt-get install default-jdk

Fordítás, futtatás terminálból:

Fordítás:


javac -d . *.java

A * helyén egy file neve is állhat, és akkor csak azt fordítja.

Futtatás:


java Main

Amennyiben Main.java file-ban van a main-ed.
