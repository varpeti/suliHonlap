Glut telepítése


sudo apt-get install freeglut3-dev

Fordítás:


g++ main.cpp -lglut -lGLU -lGL -o main.run

A main.cpp helyén más file(ok) neve(i) is állhat(nak), és akkor azt(/azokat) fordítja.

A include résznél ennek kell állnia linuxon:


#include <GL/gl.h>
#include <GL/glu.h>
#include <GL/glut.h>

windowson:


#include <gl/gl.h>
#include <gl/glu.h>
#include <gl/glut.h>
